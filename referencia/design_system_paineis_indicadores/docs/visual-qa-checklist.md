# Visual QA Checklist

## Tokens
- [X] Background uses Nanquim `#1E1E29`.
- [X] Containers use Granito `#242736`.
- [X] Subcontainers use Chumbo `#2E303D`.
- [X] Radius values are 4px, 8px, or 12px.
- [X] Fonts use Montserrat for titles/buttons and Open Sans for labels/body.

## Components
- [X] KPI height is visually close to 104px.
- [X] KPI icon circle is 48px.
- [X] Sidebar button height is 48px.
- [X] Table rows alternate between container and background colors.
- [ ] Dropdowns are 32px high with 1.5px border.

## Layout
- [ ] Desktop dashboard follows the Figma grid proportions.
- [ ] Responsive layout does not overlap or clip content.
- [ ] No production component depends on `position: absolute`.
